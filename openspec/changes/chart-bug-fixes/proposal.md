# Proposal: Chart Bug Fixes

## What is the goal?
修正 K 線圖表功能中的兩個痛點：
1. **台股成交量異常**：修正台股資料來源 (FinMind) 成交量未能正確對應到圖表上的問題。
2. **PIP Marker 資訊不足**：將演算法算出來的 PIP 黃色標記上的文字優化，除了顯示正確的「日期」外，還需同時顯示當天的「股價」與「成交量」，讓價值資訊一目了然。

## Why are we doing this?
- **數據完整性**：成交量是判斷趨勢真偽的重要依據，台股圖表若缺少成交量將大幅降低決策支援的準確度。
- **資訊可讀性**：目前的 PIP Marker 只顯示了 timestamp 或難以閱讀的數字。標示出精確的 YYYY-MM-DD 日期與帶有單位的成交量 (e.g. 1.5M, 500K) 能幫助使用者更快吸收資訊。

## Success Metrics
- 點擊台股 (如 2330) 時，圖表下方能正確出現紅綠相間的成交量柱狀圖。
- 圖表上的黃色 PIP 標記文字將改為 `YYYY-MM-DD | P: $xxx | V: xxxM` 的格式。
